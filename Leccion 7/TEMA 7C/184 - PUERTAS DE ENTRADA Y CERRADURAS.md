

Para que una puerta de entrada sea segura, debe estar equipada con una cerradura. Por lo general, una puerta de entrada segura se cierra y bloquea automáticamente en lugar de depender del usuario para cerrarla y bloquearla. Los tipos de cerradura se pueden clasificar de la siguiente manera:

- Física: son cerraduras convencionales que impiden el accionamiento de la manija de la puerta sin utilizar una llave. Los tipos más caros ofrecen una mayor resistencia contra el forzado de cerraduras.
- Electrónica: son cerraduras que, en lugar de usar una llave, funcionan al ingresar un PIN en un teclado electrónico. Este tipo de cerradura también se conoce como de cifrado, combinación o sin llave. Una cerradura inteligente se puede abrir con una tarjeta magnética o con un proximity reader (lector de proximidad) para detectar la presencia de un token físico, como un llavero inalámbrico o una tarjeta inteligente.

Vestíbulo de control de acceso (mantrap)
Un access control vestibule (vestíbulo de control de acceso), también conocida como mantrap (o trampa para hombres), es una medida de seguridad que regula la entrada a una zona segura. Se trata de dos puertas o portones que se interbloquean y permiten que solo una persona pase a la vez. La primera puerta se abre después de que se le concede acceso a la persona a través de un sistema de control de acceso, como un lector de tarjetas o un escáner biométrico. Una vez que la persona ingresa al vestíbulo, la primera puerta se cierra. La segunda puerta se abre solo cuando la primera puerta está bien cerrada. De este modo, se garantiza que una sola persona pueda entrar o salir a la vez, evitando así el acceso no autorizado o el tailgating (seguimiento). Los vestíbulos de control de acceso se utilizan con frecuencia en entornos de alta seguridad, como centros de datos, edificios gubernamentales e instituciones financieras, para ofrecer una capa adicional de control de seguridad física. Disuaden eficazmente el acceso no autorizado a áreas seguras y protegen los activos sensibles contra posibles ataques físicos.

Cerraduras de cable
Las cable locks (cerraduras de cable) se fijan a un punto seguro del chasis del dispositivo. El chasis de un servidor puede venir con un bucle metálico y una ranura de seguridad Kensington. Además de fijar el chasis a un bastidor o escritorio, la posición del punto de fijación evita que el chasis se abra sin tener que quitar primero el cable.

Credenciales de acceso
Las access badges (credenciales de acceso) son un componente fundamental de la seguridad física en las organizaciones más grandes donde es crítico tener el control del acceso a varias ubicaciones. En lugar de llaves físicas, se emiten tarjetas de plástico con bandas magnéticas, chips de identificación por radiofrecuencia (RFID) o tecnología de comunicación de campo cercano (NFC) a personas autorizadas, como empleados, contratistas o visitantes. Las credenciales de acceso sustituyen a las claves físicas, pero proporcionan acceso de manera similar. Para ello es necesario que la credencial se deslice, toque o se acerque a un lector del punto de acceso, como una puerta o un torniquete. El lector se comunica con un sistema de control para verificar la autenticidad de la credencial y el nivel de acceso otorgado al titular. Si el sistema reconoce la credencial como válida y autorizada para esa área, la puerta se desbloquea y se otorga el acceso. 

Es importante tener en cuenta que la implementación de este tipo de sistema de control de acceso requiere mecanismos magnéticos de bloqueo de puertas y lectores de tarjetas de acceso, que dependen de la energía eléctrica y las comunicaciones de red en cada punto de acceso (como una entrada).

	Un sistema de control de acceso físico (PACS) es un componente fundamental en la gestión y el mantenimiento de la seguridad dentro de una instalación. Es un sistema diseñado para controlar quién puede acceder a lugares específicos dentro de un edificio o sitio. El PACS funciona a través de una combinación de hardware y software, que incluye tarjetas o credenciales de acceso, lectores de tarjetas, paneles de control de acceso y una red de control centralizada. El sistema PACS aporta una valiosa capacidad de registro de las actividades de acceso de las credenciales.

Además de controlar el acceso, las credenciales de acceso también sirven como una forma de identificación, al mostrar información pertinente sobre el titular de la credencial, como su nombre, puesto y fotografía. De este modo, ayuda a identificar rápidamente a las personas dentro de las instalaciones y a verificar que se encuentren en un área apropiada para su función o propósito. Además, las credenciales de acceso pueden aportar datos valiosos para auditorías e investigaciones relacionadas con la seguridad. Cada vez que se utiliza una credencial, un sistema PACS puede registrar la hora, la ubicación y la identidad asociadas con el evento de acceso. Puede ser crucial para investigar las violaciones de seguridad, comprender los patrones de movimiento e incluso planificar estrategias de evacuación de emergencia.
