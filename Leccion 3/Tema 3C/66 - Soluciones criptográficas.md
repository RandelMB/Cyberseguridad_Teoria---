1.4 Explicar la importancia de usar soluciones criptográficas adecuadas. 

Como profesional en seguridad informática, debe ser capaz de seleccionar soluciones criptográficas adecuadas para un problema de seguridad determinado. Una solución criptográfica utiliza cifrados y herramientas, como certificados y firmas digitales, para implementar un control de seguridad. Los controles criptográficos se pueden utilizar para proteger archivos de datos y bases de datos, y se pueden utilizar para proteger la información que se transfiere a través de las redes.
