Las bases de seguridad de red describen un conjunto de controles y configuraciones de seguridad mínimos para una red. Las líneas de base generalmente cubren áreas como configuraciones de cortafuegos, ajustes de seguridad de enrutadores y switches, configuraciones de puntos de acceso inalámbricos y protocolos de seguridad para dispositivos de red. Las líneas de base proporcionan un punto de partida para el proceso de endurecimiento. Se aseguran de que se desactiven los servicios innecesarios, se cambien las contraseñas predeterminadas, se apliquen los protocolos de seguridad y se incluyan muchas otras configuraciones. Las herramientas compatibles con el Protocolo de automatización de contenido de seguridad (SCAP) pueden automatizar la evaluación de las configuraciones de seguridad con respecto a una línea de base definida.
