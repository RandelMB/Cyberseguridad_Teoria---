Muchos ataques de red son lanzados por hosts vulnerados que ejecutan varios tipos de código malicioso.  Los indicadores de ejecución de código malicioso son capturados por el software de protección de puntos de conexión o se descubren después del hecho en los registros de cómo el malware interactuó con la red, el sistema de archivos y el registro. Para comprender cómo y dónde se generan estos indicadores, es útil considerar los principales tipos de actividades maliciosas:

Shellcode: este es un programa mínimo diseñado para explotar una vulnerabilidad en el sistema operativo o en una aplicación legítima para obtener privilegios, o para colocar una puerta trasera en el host si se ejecuta como un troyano. Habiendo ganado un punto de apoyo, este tipo de ataque será seguido por algún tipo de conexión de red para descargar herramientas adicionales.
Volcado de credenciales: el malware puede intentar acceder al archivo de credenciales (SAM en una estación de trabajo local de Windows) o rastrear las credenciales guardadas en la memoria por el proceso del sistema lsass.exe. Además, un ataque DCSync intenta engañar a un controlador de dominio para que replique su lista de usuarios junto con sus credenciales con un host deshonesto.
Pivote/movimiento lateral/ataque interno: el procedimiento general es utilizar el punto de apoyo para ejecutar un proceso de forma remota, mediante una herramienta como PsExec o PowerShell. El atacante puede estar buscando activos de datos o puede intentar ampliar el acceso cambiando la configuración de seguridad del sistema, como abrir un puerto de cortafuegos o crear una cuenta. Si el atacante vulneró una cuenta, estos comandos pueden mezclarse con las operaciones de red ordinarias, aunque podrían ser un comportamiento anómalo para esa cuenta.
Persistencia: este es un mecanismo que permite que la puerta trasera del actor de amenaza se reinicie si el host se reinicia o el usuario se desconecta. Los métodos típicos son usar claves de ejecución automática en el registro, agregar una tarea programada o usar suscripciones de eventos de instrumentación de administración de Windows (WMI).
