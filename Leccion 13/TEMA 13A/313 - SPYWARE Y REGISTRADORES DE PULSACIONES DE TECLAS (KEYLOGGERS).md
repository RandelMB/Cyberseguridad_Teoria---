Los primeros virus y gusanos se centraron en el potencial destructivo de poder replicarse. Sin embargo, a medida que los usos rentables de este software se hicieron evidentes, comenzaron a codificarse con cargas útiles diseñadas para facilitar la intrusión, el fraude y el robo de datos. Tanto el bloatware como el malware se pueden utilizar para diferentes niveles de monitoreo:

Cookies de seguimiento: las cookies son archivos de texto plano y no son malware. Sin embargo, si se permiten según la configuración del navegador, las cookies de terceros pueden utilizarse para registrar la actividad web, rastrear la dirección IP del usuario y recopilar varios metadatos, como consultas de búsqueda e información sobre el software y la configuración del navegador. Las cookies de seguimiento son creadas por anuncios y widgets de análisis incrustados en muchos sitios web.
Supercookies y balizas web (beacons): dado que el software de navegación brinda al usuario cierto control sobre qué cookies aceptar, las empresas de marketing web idearon formas alternativas de implementar el seguimiento que son difíciles de desactivar. Una supercookie es un medio para almacenar datos de seguimiento de forma no convencional, como guardarlos en la memoria caché sin declarar que los datos son una cookie o codificar datos en solicitudes de encabezado. Una baliza web es una imagen de un solo píxel incrustada en un sitio web. Si bien es invisible para el usuario, el navegador debe realizar una solicitud para descargar el píxel y cargar el sitio, lo que le brinda al host de baliza la oportunidad de recopilar metadatos, realizar huellas digitales del navegador y, potencialmente, ejecutar secuencias de comando de seguimiento.
Adware: esta es una clase de programa potencialmente no deseado (PUP)/bloatware que realiza reconfiguraciones del navegador, como permitir cookies de seguimiento, modificar proveedores de búsqueda predeterminados, abrir páginas de patrocinadores al inicio, agregar marcadores, entre otros.  El adware se puede instalar como un programa o como una extensión o complemento del navegador.
Spyware: se trata de un tipo de malware que puede realizar un seguimiento similar al adware, pero también puede monitorear la actividad de las aplicaciones locales, tomar capturas de pantalla y activar dispositivos de grabación, como un micrófono o una cámara web.  Otra técnica de spyware consiste en realizar redireccionamientos de DNS a sitios web fraudulentos (pharming).
Keylogger: se trata de un tipo de spyware que intenta activamente robar información confidencial registrando las pulsaciones de teclas. Por lo general, el atacante espera descubrir contraseñas o datos de tarjetas de crédito.

Se utiliza la herramienta de acceso remoto Metasploit Meterpreter para volcar las pulsaciones de teclado del equipo víctima, lo que revela la contraseña utilizada para acceder a una aplicación web.

Los keyloggers no solo se implementan como software. Una secuencia de comandos maliciosa puede transmitir las pulsaciones de teclas a un sitio web de terceros. También hay dispositivos de hardware para capturar las pulsaciones de teclas en un adaptador USB modificado que se inserta entre el teclado y el puerto. Dichos dispositivos pueden almacenar datos de forma local o pueden venir con conectividad Wi-Fi para enviar datos a un punto de acceso encubierto. Otros ataques incluyen rastreadores inalámbricos para registrar datos de pulsaciones de teclas, superposición de teclados PIN de cajeros automáticos, y más.
