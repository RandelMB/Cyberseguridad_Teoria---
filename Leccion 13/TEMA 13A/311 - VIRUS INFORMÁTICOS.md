Un virus informático es un tipo de malware diseñado para replicarse y propagarse de un equipo a otro, por lo general “infectando” aplicaciones ejecutables o código de programa. Existen varios tipos diferentes de virus y en general se clasifican según los diferentes tipos de archivos o medios que pueden infectar.

No residente/infector de archivos: el virus está contenido dentro de un archivo ejecutable del host y se ejecuta con el proceso del host. El virus intentará infectar otras imágenes de procesos en el almacenamiento persistente y realizar otras acciones de carga útil. Luego devuelve el control de nuevo al programa principal.
Residente en memoria: cuando se ejecuta el archivo host, el virus crea un proceso nuevo para sí mismo en la memoria. El proceso malicioso permanece en la memoria, incluso si se termina el proceso del host.
De sector de arranque: el código del virus se escribe en el sector de arranque del disco o en la tabla de particiones de un disco fijo o medio USB y se ejecuta como un proceso residente en la memoria cuando se inicia el sistema operativo o el medio se conecta al equipo.
Virus de secuencias de comandos y macros: el malware utiliza las funciones de programación disponibles en los motores de secuencias de comandos locales para el sistema operativo o el navegador, como PowerShell, Instrumentación de administración Windows (WMI), JavaScript, documentos de Microsoft Office con código de Visual Basic para aplicaciones (VBA) habilitado o documentos PDF con JavaScript habilitado.
Además, el término “multipartito” se utiliza para los virus que utilizan múltiples vectores y el término “polimórfico” se utiliza para los virus que pueden cambiar de forma dinámica u ofuscar su código para evadir la detección.

Lo que estos tipos de virus tienen en común es que deben infectar un archivo host o un medio. Un archivo infectado puede distribuirse a través de cualquier medio normal, ya sea en un disco, en una red, como un adjunto a un correo electrónico o publicación en redes sociales, o como una descarga desde un sitio web.
