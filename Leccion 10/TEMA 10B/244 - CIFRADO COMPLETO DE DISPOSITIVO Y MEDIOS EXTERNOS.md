
Todas las versiones de los sistemas operativos de dispositivos móviles para teléfonos inteligentes y tabletas (excepto las primeras) ofrecen un cifrado del dispositivo completo. En iOS, hay varios niveles de cifrado.

- Todos los datos del usuario en el dispositivo están siempre cifrados, pero la clave se almacena en el dispositivo. Esto se utiliza principalmente como un medio para limpiar el dispositivo. Para que no se pueda acceder a los datos, el sistema operativo solo debe eliminar la clave en lugar de limpiar cada ubicación de almacenamiento.
- Los datos del correo electrónico y cualquier aplicación que use la opción “Protección de datos” están sujetos a una segunda ronda de cifrado mediante una clave derivada y protegida por la credencial del usuario. Esto proporciona protección de los datos en caso de robo del dispositivo. No todos los datos del usuario se cifran con la opción “Protección de datos”; por ejemplo, los contactos, los mensajes SMS y las fotos no se cifran.
En iOS, el cifrado de protección de datos se activa automáticamente al configurar un bloqueo con contraseña en el dispositivo. En Android, existen diferencias sustanciales en las opciones de cifrado entre versiones (source.android.com/security/encryption). A partir de Android 10, no existe el cifrado disco completo, ya que se considera demasiado perjudicial para el rendimiento. Los datos del usuario se cifran a nivel de archivo de manera predeterminada.

Un dispositivo móvil contiene una unidad de estado sólido (memoria flash) para el almacenamiento persistente de aplicaciones y datos. Algunos teléfonos Android admiten el almacenamiento extraíble a través de medios externos, como una ranura para tarjeta Micro Secure Digital (SD). Incluso pueden admitir la conexión de dispositivos de almacenamiento basados en USB. El software de cifrado del sistema operativo móvil también puede permitir el cifrado del almacenamiento extraíble, aunque no siempre es así. Se debe tener la precaución de aplicar el cifrado a las tarjetas de almacenamiento al utilizar software de terceros si es necesario, y restringir el almacenamiento de datos sensibles en ellas.

Un HSM MicroSD es un módulo de seguridad de hardware de factor de forma pequeño diseñado para almacenar claves criptográficas de forma segura. Permite utilizar el material criptográfico con diferentes dispositivos, como un equipo portátil y un teléfono inteligente.
