Deberá ser capaz de aplicar políticas y tecnologías de endurecimiento de host y evaluar los riesgos de las cadenas de suministro de terceros y de los sistemas integrados/IoT.

DIRECTRICES PARA IMPLEMENTAR SOLUCIONES DE SEGURIDAD DE HOST

Siga estas directrices para implementar o reevaluar la seguridad e integración de los puntos de conexión con sistemas integrados o de IoT:

- Establezca líneas base de configuración para cada tipo de host. Asegúrese de que los hosts se implementen conforme a la línea de base de configuración y establezca un monitoreo para garantizar el cumplimiento.
- Configure el cifrado disco completo.
- Implemente una solución de protección de puntos de conexión que cumpla con los requisitos de seguridad para funciones como antimalware, cortafuegos, IDS, EDR y DLP.
- Establezca procedimientos de administración de parches para probar las actualizaciones en diferentes grupos de hosts y garantizar la gestión tanto del sistema operativo como del software de terceros.
- Cree un plan de gestión para cualquier dispositivo IoT utilizado en el lugar de trabajo.
- Evalúe los requisitos de seguridad para los sistemas integrados ICS y/o SCADA:
	- Adquisición de sistemas integrados y IoT seguros
	- Uso de controles criptográficos para la autenticación, la integridad y la resiliencia
	- Control de acceso y segmentación para redes OT
	- Soporte del proveedor para la administración de parches
