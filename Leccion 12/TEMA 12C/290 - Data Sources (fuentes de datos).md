
OBJETIVOS DEL EXAMEN CUBIERTOS

4.9 A partir de un escenario, usar las fuentes de datos para respaldar una investigación.

Las redes, los hosts y las aplicaciones producen una gran cantidad de datos a través de diferentes mecanismos. Identificar todas estas fuentes de datos y luego escanearlas en busca de indicadores de amenazas es un desafío importante para todos los tipos de organizaciones. Como profesional de la seguridad, debe poder identificar y utilizar fuentes de datos apropiadas para realizar la respuesta a incidentes de la manera más eficiente posible.
